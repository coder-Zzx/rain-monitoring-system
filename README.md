## 前言
    由于本人常年宅寝室，偶有出门拿外卖/吃饭需求，而阳台上堆满了东西无处落脚，于是就有了本系统  
    将本系统放在阳台，实时监测是否下雨，并将数据传到电脑/上传服务器，便可足不出户知晓此时此刻是否正在下雨

## 组件
    nodeMCU，雨水传感器，OLED显示屏，温湿度监测模块
